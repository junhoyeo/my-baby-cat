# My Baby Cat
- 이미지를 포함한 모든 파일은 `Main.c`가 위치한 프로젝트 루트에서부터 절대경로로 참조됩니다.
- OS X에서 페러렐즈를 이용해 가상화된 Windows 환경에서 개발했습니다.
