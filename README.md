## 기획

### 레이아웃
- 쿠키런 같은 느낌
- 가로로 긴 스크린
- 고양이는 왼편에 고정됨(속도에 맞춰 제자리걸음)
- 배경/장애물 오른쪽에서 왼쪽으로 움직임
- 슬라이딩, 점프, 죽었을 때 스프라이트 필요

### 조작
- 공룡 게임처럼...
- 스페이스 바 눌러서 점프
- 다운 키 눌러서 슬라이딩
- 아이템은 접촉하면 바로 써짐
- 젤리(?)도 접촉하면 바로 먹는 거임

### 아이템
- 체력 회복
- 속도 감소 or 대시
- 무적
- 젤리 당 추가점수

### 기능
- 배경음악 재생
- 메인 화면(로비) 존재
- 죽고선 점수 보여주고 재시작 또는 메인 화면으로 돌아가기
- 최고 점수, 코인, 레벨 등 저장됨
